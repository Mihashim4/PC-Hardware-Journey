# PC Hardware Journey
---

## 🔧 HDD Transfer & Storage Upgrade

This project demonstrates transferring a hard disk drive (HDD) from an older computer setup to another system in order to increase storage capacity.
____________________________________________________
### 📌 What Was Done

- Removed the HDD from an old PC
- Connected the HDD to another system
- Tested drive detection
- Checked storage availability
- Demonstrated the hardware setup process

_____________________________________________________

### 💻 Topics Covered

- Computer Hardware Basics
- HDD Handling
- SATA Connections
- Storage Expansion
- PC Upgrade Process

_____________________________________________________

### 🎯 Purpose

This project was created to explore computer hardware practically and improve technical understanding of storage devices and PC components.

### ⚠️ Additional Observation

The transferred HDD already contained a Windows installation from the previous PC. This can sometimes cause boot conflicts or detection issues when connected to another system without formatting or changing boot settings.
